# 기술 스택 & 활용 능력

## 🟢 상 (능숙)
- C 언어  
  • 테트리스, 지렁이 게임 등 콘솔 기반 프로젝트 직접 구현  
  • 알고리즘 문제 풀이 경험 (백준 실버 수준)  
- 터미널  
  • CLI 환경에서 컴파일 및 실행에 익숙  
- Notion  
  • 학습 내용 및 프로젝트 기록 체계적으로 관리  

## 🟡 중 (활용 가능)
- HTML / CSS / JavaScript  
  • 코드 구조 이해 및 간단한 웹 개발 가능  
- Git / GitHub  
  • 버전 관리 및 협업 경험 보유  

## 🔵 하 (학습 중)
- Java / Python  
  • 기본 문법 이해 및 문제 풀이 가능  
  • 실전 개발 적용을 목표로 학습 중
